Virtualización | ¿Que es?
La virtualización es una tecnología que permite crear versiones virtuales de recursos informáticos, como sistemas operativos, servidores, almacenamiento y redes. En lugar de ejecutar una sola instancia física de un recurso en hardware dedicado, la virtualización permite ejecutar múltiples instancias virtuales en el mismo hardware físico. Esto se logra mediante un software llamado hipervisor o monitor de máquinas virtuales.

Virtualización | TIPOS
Virtualización de servidor: Ejecuta varios sistemas operativos en un solo servidor físico, con cada sistema operativo en su propia máquina virtual. Virtualización de escritorio: Permite a un usuario ejecutar múltiples sistemas operativos en una única estación de trabajo. Virtualización de red: Crea redes virtuales para facilitar la administración y configuración de redes. Virtualización de almacenamiento: Agrupa y administra recursos de almacenamiento de manera centralizada. Virtualización de aplicaciones: Encapsula aplicaciones y sus dependencias para facilitar la portabilidad entre diferentes entornos.

Docker | ¿Que es?
Docker es una plataforma de virtualización de contenedores que simplifica el desarrollo, implementación y ejecución de aplicaciones. Los contenedores son unidades ligeras y portátiles que incluyen todo lo necesario para ejecutar una aplicación, como el código, las bibliotecas y las dependencias. Docker utiliza la tecnología de contenedores para permitir que las aplicaciones se ejecuten de manera consistente en diferentes entornos.
